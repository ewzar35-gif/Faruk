# Faruk - Spotify Benzeri Müzik Player Uygulaması 🎵

Android'de Kotlin ve Jetpack Compose kullanarak geliştirilmiş modern bir müzik player uygulaması.

## Özellikler 🎶

- ✅ Müzik oynatma/durdurma/devam ettirme
- ✅ Şarkı listesi yönetimi
- ✅ İleri/Geri tuşları
- ✅ Ses seviyesi kontrolü
- ✅ Şarkı süresi göstergesi
- ✅ Modern UI (Jetpack Compose)
- ✅ Material Design 3

## Teknolojiler 🛠️

- **Kotlin** - Programlama dili
- **Jetpack Compose** - Modern UI framework
- **Coroutines** - Asenkron işlemler
- **MediaPlayer** - Müzik oynatma API'si
- **Gradle** - Build tool

## Kurulum 📦

### Gereksinimler
- Android Studio (Hedgehog veya üzeri)
- JDK 11+
- Android SDK 21+

### Adımlar
1. Depoyu klonlayın:
```bash
git clone https://github.com/ewzar35-gif/faruk.git
cd faruk
```

2. Android Studio'da açın

3. Uygulamayı çalıştırın:
```bash
./gradlew run
```
